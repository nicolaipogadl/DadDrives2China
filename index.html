Dad Drives To China

Sifting through old stuff together and discover something new

Download this project as a .zip file Download this project as a tar.gz file

For the longest time I wanted to ask my dad more about his travels to China in the 1980s - shortly after the government decided to open up more to the world. I knew some stories and have seen a few pictures, but it still is a mystery to me.

So with this project my dad and I would like to change that.Stretched out over the next two or three years we will try to digitize some of the old pictures, collect stories and try to connect them to present day China.

This here is just the landing page for the project - the whole documentation is available via the github wiki.
